## B. Short description (2–4 lines)

Chatvioso is a real-time team communication platform organized by workspaces. Users collaborate in direct messages, groups, and channels with live messaging, typing indicators, presence, file attachments, reactions, pins, mentions, and in-app notifications. The stack is Laravel 12 (REST API, Sanctum, Reverb) and Next.js (TypeScript, TanStack Query). The API is designed so additional clients (e.g. Flutter) can reuse it.

## C. Medium summary (one short paragraph)

Chatvioso is a full-stack, workspace-based team chat application built to demonstrate production-style SaaS and real-time systems. It offers direct messages, group conversations, and channels; real-time delivery, presence, and typing indicators via Laravel Reverb (WebSockets); and collaboration features such as file sharing, emoji reactions, pinned messages, and @mentions with in-app notifications. The backend is a Laravel 12 REST API with Sanctum auth and policy-based authorization; the frontend is Next.js with React, TypeScript, and TanStack Query. The architecture is API-first and multi-tenant by workspace, with clear separation between backend and frontend so the same API and realtime layer can support future clients (e.g. Flutter). The project showcases API design, real-time architecture, secure file handling, and end-to-end product delivery suitable for full-stack or backend-focused roles.
