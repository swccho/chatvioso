# Screenshots and presentation assets

This folder is intended for **screenshots and other visual assets** used in the README, portfolio, or other presentation materials.

Screenshots are not committed by default. Add them here when ready and reference them from the main README or docs.

---

## Naming convention

Use a consistent prefix and short description so assets are easy to find and order:

| Filename | Description |
|----------|-------------|
| `01-login.png` | Login page |
| `02-register.png` | Registration page |
| `03-chat-layout.png` | Main app layout: sidebar, chat area, right panel |
| `04-workspace-switcher.png` | Workspace switcher / sidebar context |
| `05-conversation-view.png` | Conversation with messages, composer, reactions |
| `06-settings.png` | Profile or workspace settings page |
| `07-notifications.png` | Notifications panel or dropdown |
| `08-search.png` | Search UI or search results |

Use the same base names if you add different formats (e.g. `01-login.webp`) or variants (e.g. `05-conversation-view-dark.png`).

---

## Planned views (for reference)

- **Login** — Auth form, link to register
- **Register** — Sign-up form
- **Main chat layout** — Three-column: sidebar (workspaces, conversations), center (header, messages, composer), right (shared files, pinned, or empty)
- **Workspace switcher** — Dropdown or list of workspaces in sidebar
- **Conversation view** — Messages, reactions, reply, attachments; typing/presence if visible
- **Settings** — Profile/settings or workspace settings (name, logo, members)
- **Notifications / search** — Notification panel and/or search results (users, conversations, messages)
